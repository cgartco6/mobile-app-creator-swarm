# Component Library Manager

**Standalone module for Mobile App Creator - Reusable UI components, themes, and assets management.**

## Features
- Add / search / version components (React Native + Flutter)
- Theme management
- Export to files or JSON for easy integration with ai_app_builder, ai_video_creator, etc.
- Local-first with persistent index

## Installation
```bash
cd component_library_manager
pip install -e .
